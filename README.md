## Description
<p>
  A comprehensive solution developed in C# for the operational management of retail chains. The system centralizes inventory control, warehouse movements, and sales processes, using Microsoft SQL Server to ensure data integrity.
  <br><br>
  The project architecture implements a separation of responsibilities through the "ReglasDeNegocio" library, which encapsulates all data access logic and system policies, facilitating maintenance and scalability.
</p>
